# 🎾 앵그리 테니스 클럽 대진표 생성기

클럽 주간 모임의 복식 대진표를 자동으로 만들어주는 도구입니다.

## 바로 사용하기

**`dist/앵그리대진표.html` 파일을 더블클릭**하면 브라우저에서 바로 열립니다.
설치·인터넷 연결이 필요 없고, 휴대폰으로 파일을 보내서 열어도 동작합니다.

1. 모임 유형 선택 — **정기모임**(a·b코트 게임 + c코트 레슨) 또는 **월례대회**(3코트 전부 게임)
2. 오늘 참석자 번호 선택 (남 1~9, 여 A~I — 번호가 빠를수록 실력 상위)
3. 필요하면 개인 특성(게임선호⚡/신규회원🔰/혼복선호💞)과 지각·조퇴 라운드 설정
4. **대진표 생성** — 마음에 안 들면 **다시 섞기**
5. 수동 조정: 대진표에서 선수 이름 두 개를 차례로 누르면 자리를 맞바꿈 (규칙 위반 시 경고 표시)
6. **인쇄** 버튼으로 출력

마지막에 선택한 참석자·특성은 브라우저에 저장되어 다음 주에 그대로 불러옵니다.

## 공유와 권한

데이터(명단·설정·대진표)는 각자 기기의 브라우저에 저장되며, 공유는 링크로 합니다.

- **📤 대진표 공유 링크** — 매주 단톡방에 올리는 용도. 받는 사람은 **보기 전용 화면**(대진표 + 인쇄)만 봅니다.
- **👥 명단만 공유 링크** — 명단·설정을 다른 관리자(예: 회장)에게 전달할 때 사용.
- **관리자 비밀번호** (고급 설정에서 지정) — 링크 속 명단·설정은 이 비밀번호로 암호화(AES-GCM)됩니다.
  비밀번호를 아는 사람이 보기 화면에서 "🔑 관리자 모드"로 들어가면 명단·설정·대진표가 그 기기에 저장되어
  이후 직접 대진표를 만들 수 있습니다. 매주 대진표 링크를 열기만 해도 명단 백업이 자동 갱신됩니다.
- 비밀번호를 변경하면 이전 링크로는 명단을 불러올 수 없습니다(대진표 보기는 계속 가능).
- 참고: 대진표 자체(이름 포함)는 링크를 가진 누구나 볼 수 있습니다. 명단의 실력 순위·개인 특성만 암호로 보호됩니다.

## 대진 규칙 (우선순위 순)

1. 한 라운드에 1인 1게임, 게임코트는 항상 채움 **(하드)**
2. 잡복 금지 — 남복/여복/혼복만 **(하드)**
3. 같은 파트너 중복 0회 **(하드)**
4. 게임 수 균등(차이 ≤1), 2라운드 연속 결장 금지
5. 정기모임: 3라운드 내 전원 1회 레슨 로테이션
6. 월례대회: 참가자 1인당 최소 1게임 혼복
7. 빡겜 라운드(설정, 기본 1·2·3): 비슷한 실력 4명이 한 게임 — 특히 **2라운드는 남복/여복 상위 랭커(1~4위)끼리 우선 편성**
8. 같은 상대 재대면 최소화(상한 설정, 기본 2번) + 팀 합산 점수 균형
9. 개인 특성 반영 (정기모임)
10. 게임 유형 — 혼복 선호 라운드(설정, 기본 1·3)는 혼복 위주, 나머지는 동성복식 위주. 월례는 동성복식 위주
11. 레슨 그룹 실력 유사성

지킬 수 없는 조합(예: 남자 1명, 12명 월례에 남자 홀수)은 이유와 대안을 안내하고,
불가피한 완화(연속 결장 등)는 파란 배너로 표시됩니다.

## 개발

순수 JavaScript(ESM), 빌드 도구·외부 의존성 없음. 개발에만 Node.js 18+ 필요.

```
npm test        # 8~18명 남녀 전 조합 + 엣지 케이스 자동 검증 (139 tests)
npm run build   # dist/앵그리대진표.html 단일 파일 생성
```

```
engine/
  scheduler.js   공개 API: generateSchedule(config)
  planner.js     라운드·코트 계획, 실현 가능성 사전 검사 (패리티/파트너 풀)
  construct.js   라운드별 구성: 결장자 선정 → 페어링 백트래킹 → 매치업
  optimize.js    사전식 비용 튜플 + 로컬 서치 (하드 제약 보존 스왑)
  validate.js    검증기 — 엔진 자기검증·UI 수동편집 경고·테스트 공용
  rng.js         시드 PRNG (같은 시드 = 같은 대진표)
app.js           웹 UI (index.html은 로컬 서버에서 모듈로 로드)
build.mjs        엔진+UI를 단일 HTML로 번들
test/            node --test (파일은 .mjs — 일부 윈도우 환경에서 한글 경로 .js 엔트리 크래시 회피)
```
